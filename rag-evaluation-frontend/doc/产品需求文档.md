# RAG评测网站产品需求文档

## 1. 主题
RAG系统评测平台：简化RAG准确率评估与数据准备的一体化平台

## 2. 背景与问题
RAG评测成本高、标准不统一、覆盖不足、结果难分析，导致迭代缓慢。

## 3. 目标
- 将评测数据准备时间降低约80%
- 提供统一、多维度评测标准与可解释报告
- 支持不同领域与场景的快速评测与对比

## 4. 目标用户与需求
- 数据科学家/AI工程师：快速评测、定位弱项、迭代验证
- 产品经理：可视化报告、趋势追踪、可对外沟通
- 领域专家：无技术门槛参与评测与专业标准定义

## 5. 产品范围（核心能力）
### 5.1 数据集与问答对
- Excel/CSV导入导出，标准模板
- 基于知识库的自动问答生成（按难度/类型配置），支持人工审核

### 5.2 评测引擎
- 自动评测：多模型评分，多维度（准确性/相关性/完整性/简洁性），支持权重与评分规则
- 人工评测：批量与单条深度评测，注释与引用核验
- 混合评测：自动先行、人工复核，输出对比与差异

### 5.3 评测管理与协作
- 多评测人分配、进度追踪、一致性分析
- 争议标记与讨论

### 5.4 分析与报告
- 仪表盘与趋势分析
- 错误类型识别与改进建议
- 导出报告
